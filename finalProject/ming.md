# 项目名称
 
> MING  共识平台

## 💻 项目 Repo

https://github.com/LuckeeDAO/ming.git


## 📌 项目简介

基于 Solana 的 NFT 命理共振平台，依托区块链不可篡改的共识机制，构建了一个以 NFT 为载体的“人类共识能量库”。通过铸造、持有与交互 NFT，用户可触发链上命理祝福，实现天地人三才能量的数字共振。

我们致力于为探寻命理学底层逻辑的当代人，提供一种可参与、可验证、可迭代的“人文科技实验场”，在尊重传统智慧的基础上，探索人与时空能量互动的新范式。

## 🛠️ 技术栈
 
- 智能合约：Rust + Anchor Framework
- 前端：Next.js + TypeScript + Wallet Adapter
- 工具：Solana CLI, @solana/web3.js
